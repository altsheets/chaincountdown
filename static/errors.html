These urls return different specific error messages that are caught:

TYPE TEXT: Designing the url

If you ask for the impossible, exceptions are thrown = Text is returned.
  • http://localhost:8888/nxt/ and http://localhost:8888/nxt/1000000 and http://localhost:8888// and http://localhost:8888/// and or any other wrongly formatted url
  • http://localhost:8888/wrongcoin
  • http://localhost:8888/nxt/abc.png
  • http://localhost:8888/nxt/1000000.tif
  • http://localhost:8888/nxt/1000000.gif?font=aniceone
  • http://localhost:8888/nxt/1000000.gif?fontsize=20.
  • http://localhost:8888/nxt/1000000.gif?color=red
  • http://localhost:8888/nxt/10000000000000000.png

TYPE IMAGE: Transient problems ...

... are ending up as an image showing the error.

  • http://localhost:8888/nxt/0.png
  • if the apiserver is offline
  • if the apiserver is not fully synced yet
  • if the apiserver returns problems
